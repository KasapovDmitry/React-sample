- добавление поста;
- удаление поста;
- сортировка списка;
- фильтрация списка;
- модальное окно;
- компоненты (кнопки, инпут, селект и т.д.)
- получение списка постов по API;
- авторизация (front-end);
- постраничная навигация;
- подгрузка постов;
- декомпозиция;
- кастомные хуки;
- навигация.
